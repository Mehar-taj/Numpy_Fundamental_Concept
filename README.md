## Overview
This repository contains NumPy basics, practice programs, and example files created while learning NumPy in Python.

It includes:
- NumPy Introduction
- Random Variables
- A-Range Functions
- Array Indexing
- Array Slicing
- Integer Array Indexing
- Boolean Array Indexing
